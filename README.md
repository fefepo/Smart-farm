## 📌 프로젝트 개요
센서를 기반으로 작물의 **온도, 습도, 조도 환경**을 측정하고,  
임계값을 벗어나면 **장치를 자동 제어**하여 최적의 생장 환경을 유지하는 스마트 팜 시스템입니다.

## 📅 개발 기간
2023.11 ~ 2023.11

## 담당 역할
- 아두이노 기반 제어 로직 구현
- 센서 데이터 수집 및 디버깅
- 스마트 팜 환경 구성 및 장치 배치

## ⚙️ 주요 기능
- 온습도 센서(DHT22), 조도 센서를 통한 환경 모니터링
- 센서 값에 따른 자동 제어
  - 온도 → 열선패드 / 냉각 팬
  - 습도 → 가습기 모듈
  - 조도 → LED 모듈
  - LCD를 통한 실시간 환경 데이터 시각화

## 시스템 구성
- **Arduino 1 (센서 담당)**
  - 온도, 습도, 조도 측정
  - 측정 데이터를 Serial 통신으로 전송

- **Arduino 2 (제어 담당)**
  - 센서 데이터 수신
  - 조건에 따른 장치 자동 제어
  - 릴레이 모듈을 통해 고전력 장치 구동

## 사용 기술 및 장비
- Arduino Uno (2개)
- DHT22 온습도 센서
- 조도 센서
- LED 모듈
- 열선패드
- 냉각 팬
- 가습기 모듈
- 4채널 릴레이 모듈 (고전력 장치 제어)
- 브레드보드
- I2C LCD 모듈
  
## 동작 로직
- 온도 < 18℃ → 열선패드 ON
- 온도 > 25℃ → 냉각 팬 ON
- 습도 < 40% → 가습기 ON
- 습도 > 60% → 가습기 OFF
- 조도 < 500 → LED ON

## 특징
- Serial 통신 기반 **이중 아두이노 구조**
- 릴레이 모듈을 이용한 **고전력 장치 제어**
- 실시간 센서 데이터 기반 **자동화 시스템**
- LCD를 통한 사용자 **인터페이스 제공**
